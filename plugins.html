Plugins

This is an official and comprehensive list of Anypaste plugins. You're a plugin author? Contribute to this list on the anypaste-website repo.

Videos

Sendvid

Built-In
Authenticationless
Supports deletion and metadata editing for uploaded videos.
Exact plugin name: sendvid
Configuration: None
Sendvid Homepage

Streamable

Built-In
Requires Authentication
Super simple, Has ads. A standard Streamable account is required for authentation.
Exact plugin name: streamable
Configuration:
  • streamable_email: Streamable account email
  • streamable_password: Streamable account password

Streamable Homepage

Gfycat

Built-In
Authenticationless
Meant for gifs, so strips audio and truncates videos beyond 60 seconds.
Exact plugin name: gfycat
Configuration:
  • gfycat_duplicates: Set to non-empty string to enable duplicate-checking (if your video is already on gfycat, you'll get the link to the existing one).

Gfycat Homepage

Documents

Docdroid

Built-In
Requires Authentication
Very nice, automatically converts documents for cross-platform viewing. Needs an API key to function.
Exact plugin name: docdroid
Configuration:
  • docdroid_access_token: API access token for docdroid. With an account, go to settings->API->Create New Token, and enable the upload_documents scope.

Docdroid Homepage
